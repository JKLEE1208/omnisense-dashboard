                       OmniSense Dashboard
[React v19] | [Vite v7] | [Tailwind v3.4] | [Status: Active]

1.环境要求

  Node.js: 版本需 >= 18.0.0
  npm: 版本需 >= 9.0.0

2.快速开始 

第一步：进入项目根目录：

  cd omnisense-dashboard

第二步：安装依赖 

  npm install


第三步：启动本地开发环境：

  npm run dev

启动成功后，控制台将输出访问地址（默认 http://localhost:5173 ），在浏览器中打开即可。
